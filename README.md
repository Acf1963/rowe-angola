View your app in AI Studio: https://ai.studio/apps/0a3e08b2-5e69-4d54-9168-d80cd2e811ef

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

# ROWE Oil Angola - Landing Page Oficial

Este repositório contém o código-fonte da landing page da **ROWE Oil** para o mercado angolano. O projeto foi desenhado com foco em **DX (Developer Experience)** e **UX (User Experience)** de alta performance.

## 🚀 Tecnologias
- **Frontend:** HTML5 Semântico e Tailwind CSS.
- **Background:** Animação procedimental via CSS (Simulação de viscosidade de óleo).
- **i18n:** Suporte multi-língua nativo (PT, EN, GER, FR).
- **Deploy:** Automatizado via GitHub Actions.

## 🛠️ Destaques Técnicos
- **Performance:** Otimizada para redes móveis 3G/4G em Angola.
- **Autoridade:** Destaque para homologações oficiais **BMW** e **Porsche**.
- **Segurança:** Formulário de contacto integrado e mapa de localização.

## 🏁 Como visualizar
A página está otimizada para ser visualizada no **Vs Code** ou diretamente via **GitHub Pages**.
